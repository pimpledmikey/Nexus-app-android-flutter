Política de Privacidad

Última actualización: 30 de noviembre de 2025

Esta Política de Privacidad describe cómo Corporativo DRT ("nosotros", "nuestro" o "la empresa") recopila, utiliza y protege la información personal de los usuarios de la aplicación Nexus Goll ("la aplicación" o "el servicio").

Al utilizar nuestra aplicación, usted acepta las prácticas descritas en esta política. Le recomendamos leer este documento detenidamente.

1. Información que Recopilamos

1.1 Información de identificación personal

  • Credenciales de acceso: Nombre de usuario y contraseña proporcionados por la empresa para acceder al sistema.
  • Datos del empleado: Nombre completo, número de empleado, sucursal asignada y otros datos laborales sincronizados desde el sistema corporativo.

1.2 Datos de ubicación

  • Ubicación GPS: Recopilamos datos de geolocalización al momento de realizar registros de entrada/salida para validar que el usuario se encuentra en una zona autorizada (geocerca).
  • Historial de ubicaciones: Las coordenadas se almacenan junto con cada registro de asistencia.

1.3 Datos biométricos

  • Autenticación biométrica: La aplicación puede utilizar huella dactilar o reconocimiento facial para verificar la identidad del usuario. Estos datos biométricos son procesados localmente en el dispositivo y nunca se transmiten a nuestros servidores.

1.4 Imágenes y multimedia

  • Fotografías: Se captura una foto del usuario al momento de cada registro como evidencia de asistencia.
  • Códigos QR: La aplicación escanea códigos QR para facilitar el proceso de registro.

1.5 Información del dispositivo

  • Identificador del dispositivo: Modelo, sistema operativo y versión.
  • Información de red: Tipo de conexión (WiFi/datos móviles).
  • Detección de seguridad: Verificamos si el dispositivo tiene VPN activa o aplicaciones de GPS falso para prevenir fraudes.

2. Cómo Utilizamos la Información

Utilizamos la información recopilada para:

  • Gestionar y verificar los registros de asistencia de los empleados.
  • Validar que los registros se realicen dentro de las zonas geográficas autorizadas.
  • Prevenir fraudes en el control de asistencia.
  • Generar reportes de asistencia para la administración.
  • Mantener la seguridad e integridad del sistema.
  • Cumplir con obligaciones legales y laborales.

3. Almacenamiento y Seguridad de Datos

3.1 Almacenamiento

  • Los datos se almacenan en servidores seguros del Corporativo DRT.
  • La aplicación mantiene una base de datos local encriptada para funcionamiento offline.
  • Los datos locales se sincronizan automáticamente cuando hay conexión a internet.

3.2 Medidas de seguridad

  • Conexiones cifradas mediante HTTPS/TLS.
  • Autenticación segura con tokens de sesión.
  • Almacenamiento local protegido.
  • Detección de dispositivos comprometidos (root/jailbreak, VPN, GPS falso).

4. Compartición de Datos

No vendemos ni compartimos su información personal con terceros para fines comerciales o publicitarios.

Sus datos pueden ser compartidos únicamente:

  • Con el departamento de Recursos Humanos y supervisores autorizados del Corporativo DRT.
  • Con autoridades competentes cuando sea requerido por ley.
  • Con proveedores de servicios tecnológicos que nos ayudan a operar el sistema, bajo estrictos acuerdos de confidencialidad.

5. Retención de Datos

Conservamos los registros de asistencia y datos asociados durante el tiempo que sea necesario para:

  • Cumplir con la relación laboral vigente.
  • Satisfacer obligaciones legales y fiscales.
  • Resolver disputas laborales si las hubiera.

Los datos se eliminan de forma segura cuando ya no sean necesarios para estos fines.

6. Sus Derechos

Como usuario y empleado, usted tiene derecho a:

  • Acceso: Solicitar información sobre los datos que tenemos sobre usted.
  • Rectificación: Corregir datos inexactos o incompletos.
  • Información: Conocer cómo se procesan sus datos.

Para ejercer estos derechos, contacte al departamento de Recursos Humanos o Sistemas del Corporativo DRT.

7. Permisos de la Aplicación

La aplicación solicita los siguientes permisos del dispositivo:

  • Ubicación: Para validar geocercas y registrar coordenadas.
  • Cámara: Para capturar fotos de evidencia y escanear códigos QR.
  • Biometría: Para autenticación segura (opcional).
  • Internet: Para sincronizar datos con el servidor.
  • Almacenamiento: Para guardar datos localmente en modo offline.

8. Uso Corporativo

Esta aplicación es de uso exclusivo para empleados del Corporativo DRT. El uso está sujeto a:

  • El reglamento interno de trabajo.
  • Las políticas de uso de tecnología de la empresa.
  • El contrato laboral vigente.

9. Cambios a esta Política

Podemos actualizar esta Política de Privacidad periódicamente. Los cambios significativos serán notificados a través de:

  • Aviso dentro de la aplicación.
  • Comunicación por parte de Recursos Humanos.

La fecha de última actualización se indica al inicio de este documento.
