This letter serves as official documentation to certify that the application "Nexus MK" is a legitimate enterprise workforce management tool developed for internal corporate use.
1. APPLICATION PURPOSE
The Nexus MK application was developed exclusively for enterprise employee attendance management with the following functionality:
  • Employee clock-in/clock-out with photo verification
  • GPS geofencing to verify employees are at designated work locations
  • Digital attendance history for payroll processing
  • Remote registration for field employees
2. POLICY COMPLIANCE - isMonitoringTool DECLARATION
We have properly implemented the required metadata declaration in ALL AndroidManifest files as per Google Play's monitoring app policy:
<meta-data
    android:name="isMonitoringTool"
    android:value="enterprise_management" />

Declared in: main/AndroidManifest.xml, debug/AndroidManifest.xml, profile/AndroidManifest.xml
3. EMPLOYEE CONSENT AND TRANSPARENCY
We confirm that all employees using this application:
  • Have been fully informed about the monitoring capabilities during employee onboarding
  • Have provided explicit consent as part of their employment agreement
  • Are aware that attendance and location data is collected during work hours
  • Can access and view their own attendance records through the application
4. WHY THIS APPLICATION IS NOT STALKERWARE
This application does NOT qualify as stalkerware because:
  • It is NOT designed for covert or unauthorized surveillance
  • It is NOT available for public download - restricted to corporate employees only
  • It does NOT operate secretly - employees are fully aware of its functionality
  • Data collection is LIMITED to work-related purposes only
  • It is used by an employer to manage employees, NOT by individuals to track others without consent
